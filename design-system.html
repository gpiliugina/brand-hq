Signature rules — never break these

✓ Display serif is only for the hero H1. Every other piece of type uses Inter.
✓ Accent #E8FE67 is used for the stroke highlight, selected-state rings, and widget preview accent fills (progress ring arc, email signature divider, brand checker dots).
✓ Body default is 13px. One page background: #ECEDEF with the dotted radial pattern.
✓ One dark: var(--ink) #2F3A45 for text, avatars, and filled buttons. No second darker shade; no pure black.
✗ Never use the display serif anywhere other than the one H1 per page.
✗ Never use the accent as a general background or decorative colour outside its defined uses. Accent fills are limited to specific widget preview graphic elements — not chrome, not buttons, not cards.
✗ Never add gradients, vibrant chrome, or extra accent colours. The brand speaks through restraint.
✗ Never use raw hex codes or inline px values. Only reference tokens defined in :root.
